# 모델 훈련 및 분석 스크립트 설명

이 폴더는 'AI 도로 파손 분석기'에 사용될 최적의 모델을 찾기 위한 모든 훈련 및 분석 스크립트를 포함합니다.

## 파일 구성

### 1. `1_run_training_baseline.py`
- **목표:** 모든 성능 비교의 기준점이 될 기본(Baseline) 모델을 훈련합니다.
- **주요 로직:** `YOLOv8s` 모델을 `30 Epochs` 동안 기본 설정으로 훈련하여, 초기 성능을 측정합니다.

### 2. `2_run_training_over_tuned.py`
- **목표:** 다양한 데이터 증강 기법을 최대한 활용했을 때의 성능 변화를 확인하는 실험을 수행합니다.
- **주요 로직:** `100 Epochs` 동안, YOLOv8이 제공하는 거의 모든 데이터 증강(회전, 이동, 크기 조절, 상하/좌우 반전, Mosaic, Mixup 등) 옵션을 활성화하여 모델을 훈련합니다.

### 3. `3_run_training_refined_tuned.py`
- **목표:** '도로 파손'이라는 문제의 특성을 고려하여, 더 현실적이고 정제된 튜닝 전략의 효과를 검증합니다.
- **주요 로직:** `50 Epochs` 동안, 상하 반전이나 과도한 왜곡처럼 비현실적인 증강은 제외하고, 회전, 이동, 좌우 반전 등 꼭 필요한 증강 기법만 선별적으로 적용하여 모델을 훈련합니다.

### 4. `4_analyze_results.py`
- **목표:** 위 3가지 실험 결과를 하나의 그래프로 시각화하여, 어떤 튜닝 전략이 가장 효과적이었는지 객관적으로 비교 분석합니다.
- **주요 로직:** `pandas`로 각 실험의 `results.csv` 파일을 읽어온 뒤, `matplotlib`을 사용하여 3개 모델의 `mAP`, `Precision`, `Recall` 점수 변화를 하나의 종합 그래프로 시각화합니다.
