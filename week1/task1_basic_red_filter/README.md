# 1주차 과제: 기본 업무 - 특정 색상 필터링

## 목표
OpenCV의 색상 공간 변환 및 마스킹 기법을 활용하여, 이미지 내에서 특정 색상 영역을 정확히 검출하고 배경과 분리하는 필터를 구현합니다.

## 전처리 과정

1.  **이미지 로드 (Image Loading)**
    - `cv2.imread()`: 디스크의 이미지 파일을 프로그램이 처리할 수 있는 Numpy 배열 형태로 불러옵니다.

2.  **색상 공간 변환 (BGR → HSV)**
    - `cv2.cvtColor()`: 조명 변화에 강인하게 색상(Hue) 정보만을 추출할 수 있도록 BGR 색상 공간을 HSV로 변환합니다.

3.  **마스크 생성 (Mask Generation)**
    - `cv2.inRange()`: 지정된 빨간색의 HSV 범위에 포함되는 픽셀만 흰색으로, 나머지는 검은색으로 만들어 대상 영역을 표시하는 마스크를 생성합니다.

4.  **비트와이즈 연산 (Bitwise AND)**
    - `cv2.bitwise_and()`: 원본 이미지에 마스크를 적용하여, 마스크의 흰색 영역에 해당하는 픽셀만 남기고 나머지는 모두 제거합니다.

5.  **결과 저장 (Save Result)**
    - `cv2.imwrite()`: 처리된 최종 결과 이미지를 파일로 저장하여 작업을 마무리합니다.
